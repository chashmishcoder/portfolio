# omkars.dev — Personal Portfolio

My personal portfolio site. Built to show work, not just list it.

## Stack

- **Next.js 16** — App Router, static export
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion v12** — animations
- **Space Grotesk + JetBrains Mono** — typography

## Sections

Hero · About · Experience · Projects · Publications · Achievements · Recommendations · Education · Writing · Contact

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

## Deploy

Deployed on Vercel at [omkars.dev](https://omkars.dev)
